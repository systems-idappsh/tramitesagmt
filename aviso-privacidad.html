Aviso de privacidad

Documento temporal para version de prueba. Debe revisarse y aprobarse antes de produccion.

Responsable

SAGMT es responsable del tratamiento de los datos enviados voluntariamente por canales de contacto como WhatsApp, llamada, correo o redes sociales.

Datos tratados

Podran tratarse nombre, datos de contacto, descripcion del tramite y documentos compartidos por el usuario cuando sean necesarios para orientar el caso.

Finalidad

Los datos se usan para responder solicitudes, orientar servicios, dar seguimiento a consultas y mantener comunicacion relacionada con tramites o asesoria.

Conservacion y seguridad

La informacion debera conservarse solo durante el tiempo necesario para la atencion del caso, evitando solicitar datos sensibles que no sean indispensables.

Derechos y contacto

Para solicitar informacion sobre datos personales, escribe a tramitesagmt@gmail.com o contacto.sagmt@gmail.com.
